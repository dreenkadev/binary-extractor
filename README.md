# Binary Extractor

Binary file string and data extractor.

Ekstraktor string dan data dari file biner.


## Description

Extracts ASCII/Unicode strings, URLs, IPs, and emails from binary files.

## Deskripsi

Mengekstrak string ASCII/Unicode, URL, IP, dan email dari file biner.


## Installation

- Navigate to the project folder
- No dependencies required

## Instalasi

- Masuk ke folder proyek
- Tidak perlu dependensi


## Usage

- Extract: python binary_extract.py binary.exe
- Demo: python binary_extract.py --demo

## Penggunaan

- Ekstrak: python binary_extract.py binary.exe
- Demo: python binary_extract.py --demo
